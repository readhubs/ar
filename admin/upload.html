دليل إضافة كورس جديد

هذه الصفحة دليل عمل — لا تحتاج لأي سيرفر. الكورسات تُضاف يدوياً عبر GitHub.

1 ارفع ملف txt في مجلد courses-raw

عبر واجهة GitHub على الويب:

  1. افتح الريبو على GitHub
  2. انتقل لمجلد courses-raw/
  3. اضغط Add file → Upload files
  4. ارفع ملف txt باسم [slug-الكورس].txt
    مثال: negotiate-your-salary.txt
  5. اضغط Commit changes
# التنسيق الصحيح لاسم الملف: # حروف إنجليزية صغيرة + شرطة فقط — لا مسافات، لا رموز خاصة # صحيح: negotiate-your-salary.txt # صحيح: productivity-masterclass.txt # خطأ: Negotiate Your Salary.txt # خطأ: productivity_masterclass.txt

2 أضف بيانات الكورس في courses-data.js

استخدم الأداة أدناه لتوليد كود JavaScript الجاهز، ثم انسخه وألصقه في ملف courses-data.js داخل المصفوفة courses.

مولّد كائن الكورس

3 تحقق من الكورس

بعد rفع الملف وتحديث courses-data.js، انتظر 1-2 دقيقة لتحديث GitHub Pages، ثم:

  1. افتح course.html?course=[slug] للتحقق من صفحة الكورس
  2. افتح article.html?course=[slug]&article=1 للتحقق من أول مقال
  3. تأكد من ظهور العنوان والمحتوى بشكل صحيح
  4. لتفعيل الفهرسة: غيّر status من draft إلى published

📄 تنسيق ملف txt

يجب أن يتبع الملف هذا التنسيق بدقة:

[SECTION:UDEMY_DATA] TITLE: عنوان الكورس DESCRIPTION: وصف تفصيلي للكورس KEYWORDS: كلمة1, كلمة2, كلمة3 AUDIENCE: الجمهور المستهدف من الكورس REQUIREMENTS: المتطلبات المسبقة OUTCOMES: - النتيجة الأولى - النتيجة الثانية - ...الى 8 نتائج [/SECTION] [SECTION:OUTLINE] [PART:1] TITLE: عنوان الجزء الأول LECTURES: 1,2,3,4,5,6 [/PART] [PART:2] TITLE: عنوان الجزء الثاني LECTURES: 7,8,9,10,11,12 [/PART] [PART:3] TITLE: عنوان الجزء الثالث LECTURES: 13,14,15,16,17,18 [/PART] [PART:4] TITLE: عنوان الجزء الرابع LECTURES: 19,20,21,22,23,24 [/PART] [/SECTION] [SECTION:LECTURES] [LECTURE:1] TITLE: عنوان المحاضرة DURATION: 12:30 PART: 1 [SCRIPT:START] نص المحاضرة المفصّل هنا. يجب أن يكون شاملاً وغنياً بالمعلومات لأنه المصدر الذي تُبنى عليه المقالات الثمانية. [SCRIPT:END] [SLIDES:START] [SLIDE:1] TITLE: عنوان الشريحة الأولى - نقطة أولى - نقطة ثانية - نقطة ثالثة [/SLIDE] [SLIDE:2] TITLE: عنوان الشريحة الثانية - نقطة أولى - نقطة ثانية [/SLIDE] [/SLIDES] [EXERCISE:START] TITLE: عنوان التمرين العملي - الخطوة الأولى - الخطوة الثانية - الخطوة الثالثة [EXERCISE:END] [/LECTURE] [LECTURE:2] ... (24 محاضرة إجمالاً) [/LECTURE] [/SECTION]
